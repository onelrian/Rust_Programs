### Rust Statistics: Mode, Mean, and Median Calculator

A simple Rust program that computes the mode, mean, and median of a given set of numbers.
___
Features
    •    Calculates the mean (average)
    •    Finds the median (middle value)
    •    Determines the mode (most frequently occurring number)
_ _ _

> ### Cloning

Clone the repository using:

```
git clone <https://github.com/onelrian/Rust_Programs/tree/main/stats.git>
```

#### Usage

Run the program with:

cargo run

You will be prompted to enter a list of numbers. The program will then display the mean, median, and mode.
